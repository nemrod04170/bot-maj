
# BOT SCALPING ASYNCHRONE (ASYNCIO)

Ce bot analyse toutes les paires en parallèle grâce à asyncio :
- Rapidité maximale même sur des centaines de paires !
- Latence minimale sur les signaux de pump/scalping.
- Compatible avec tous les filtres, chandelles, GUI, etc.
- À lancer exactement comme avant.

Aucun changement de config n'est requis.
